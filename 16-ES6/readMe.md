## 2_example
    常量  const

    用来保存不改变的变量。若更改值，不同浏览器的提示不同。

    const a = 1;
    a 不能被修改。
    const obj = {
        a : 1,
        b : 2,
        c : {
            a : 1,
            b : 2
        }
    }

    obj.a = 10; 这个是可以的。
